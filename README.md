# x_zero_game
Игра «Крестики-нолики» против непобедимого бота с подсчётом статистики.

## Что умеет
- Игра разбита на меню и поле: можно начинать партии, возвращаться в меню и выходить из игры.
- ИСК искусственный интеллект использует алгоритм minimax, поэтому бот никогда не проигрывает и старается выиграть или свести к ничье.
- Статистика счетчиков поражений и ничьих сохраняется в файле `stats.json`, поэтому она переживает перезапуск приложения.

## Требования
- Python 3.11 или новее
- `pygame`

## Установка
1. Создайте виртуальное окружение (например, `python -m venv .venv`).
2. Активируйте его (`.venv\\Scripts\\Activate` на Windows).
3. Установите зависимость: `pip install pygame`

## Как запустить
```bash
python main.py
```

## Управление
- В меню кликайте левой кнопкой мыши по кнопкам **Играть** или **Выход**.
- В игре выбирайте пустую клетку левой кнопкой мыши, чтобы сыграть за крестики (X).
- После завершения партии можно нажать **Сыграть снова** или **В меню**.

## Статистика
После каждого поражения бота в `stats.json` увеличивается счётчик `losses`, после ничьей — `draws`. Файл создаётся автоматически при первом запуске, а при повреждении или отсутствии файла данные сбрасываются на `0`.
